# Excel Sales Dashboard 📊

## 📌 Project Overview

This project demonstrates a **data analysis and visualization dashboard built using Microsoft Excel**. The dashboard helps analyze sales data and provides clear insights into business performance through charts, pivot tables, and interactive visuals.

The goal of this project is to transform raw data into meaningful information that supports decision-making.

## 🎯 Objectives

* Analyze sales data effectively
* Create interactive dashboards in Excel
* Identify sales trends and performance metrics
* Present insights using visual charts and tables

## 📂 Project File

* `PArth_Excel_project.xlsx` – Excel dashboard and data analysis file

## 📊 Dashboard Features

* Sales performance analysis
* Pivot tables for data summarization
* Interactive charts and graphs
* KPI metrics for quick insights
* Data visualization using Excel tools

## 🛠️ Tools & Technologies

* **Microsoft Excel**
* Pivot Tables
* Data Visualization
* Charts & Graphs
* Data Analysis

## 🚀 How to Use

1. Download the `.xlsx` file from this repository.
2. Open the file in **Microsoft Excel**.
3. Navigate through the sheets to explore the dashboard and analysis.
